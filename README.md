# go-less
A pager insipred by less written in go

HOW TO COMPILE:

"$: go build goless"
HOW TO INSTALL:
"$: chmod +x install.sh"
"$: ./install.sh"
Program will be built and copied to /usr/bin 
HOW TO USE:
Either execute the program with a file as the arguments or pipe input to it 

example:
  "$: goless test.txt"
  "$: cat test1.txt test2.txt | goless"
